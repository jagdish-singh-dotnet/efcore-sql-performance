# EF Core & SQL Performance Patterns

This repository contains practical examples and patterns focused on improving
database performance when working with Entity Framework Core and SQL Server.

## Topics Covered
- Index usage and query optimization
- Common EF Core performance pitfalls
- Efficient pagination and filtering techniques
- Reducing unnecessary database round-trips
- Writing readable yet performant queries

## Scope
- Focused on real-world scenarios
- No UI or front-end components
- No production or proprietary data

## Notes
Examples in this repository are intentionally simple and designed for
learning, demonstration, and interview discussions.

## Purpose
Created to showcase practical understanding of database performance
considerations in enterprise applications.
